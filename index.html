SIGN LANGAUGE bridge

In my project, I aimed to bridge the communication gap between hearing-impaired individuals and the general population. I developed an artificial intelligence assistant with three core modules, including a Text-to-sign module for converting written text and audio into sign language, a Sign-to-text module for translating sign language into written text, and a web interface using Django to recognize American Sign Language (ASL) hand motions and convert them into speech and text. Leveraging deep learning, including convolutional neural networks (CNNs), transfer learning, and natural language processing, I used technologies such as Python, CSS, HTML, OpenCV, TensorFlow, Keras, NumPy, MediaPipe, and NLTK to create this innovative solution.

potato disease detection

This project aims to classify images of potato plants into three categories: "Potato___Early_blight," "Potato___Late_blight," and "Potato___healthy" using convolutional neural networks (CNNs). The trained model can help in the early detection and diagnosis of potato plant diseases, enabling timely interventions for crop protection and improved yield

sales data anaysis using python

This repository contains Python code for analyzing sales data using pandas, numpy, matplotlib, and seaborn. The code begins by cleaning and preparing the dataset, including data type conversion and handling missing values.
